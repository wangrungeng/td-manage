# TDengine 可视化管理工具技术说明

## 实际改动

- 新增 monorepo 基础配置。
- 新增 Fastify 后端服务。
- 新增 SQLite 表结构初始化逻辑。
- 新增 JWT 登录、多用户、多角色权限控制。
- 新增 TDengine 连接配置管理和加密保存。
- 新增 TDengine 元数据、查询、新增、覆盖式编辑、删除和 SQL 控制台接口。
- 新增完整审计日志。
- 新增 Vue 前端管理页面。
- 补强写操作失败审计、删除/编辑执行前复核、本地密钥自动生成和 TDengine 连接器资源释放。

## 关键逻辑

- `apps/server/src/db/database.ts`：创建 SQLite 表和默认角色。
- `apps/server/src/plugins/auth.ts`：JWT 鉴权和权限中间件。
- `apps/server/src/modules/tdengine/sql-utils.ts`：SQL 生成、标识符校验和风险识别。
- `apps/server/src/modules/tdengine/tdengine.routes.ts`：TDengine 查询和写操作接口。
- `apps/server/src/modules/tdengine/tdengine.client.ts`：基于 `WSConfig + sqlConnect` 的 TDengine WebSocket 客户端封装。
- `apps/web/src/views/DataBrowserView.vue`：数据浏览、新增、编辑、删除页面。

## 影响范围

- 当前项目为新项目，本次新增前后端和文档文件。
- 不涉及其他云平台项目源码。
- 不涉及真实服务器或真实数据库连接。
