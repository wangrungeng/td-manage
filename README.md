# td-manage

`td-manage` 是一个本地 Web 版 TDengine 可视化数据管理工具，面向个人开发和数据排查场景。
## 中文/英文
## 功能范围

- 本地登录、多用户、多角色权限控制。
- SQLite 保存系统账号、连接配置和审计日志。
- TDengine 连接管理、连接测试、库表元数据浏览。
- 表数据分页查询，默认要求时间范围，避免无边界扫描。
- 数据新增、覆盖式编辑、精确时间戳删除，执行前必须 SQL 预览和二次确认。
- 写操作完整审计。

## 安全边界

- 默认只监听 `127.0.0.1`。
- TDengine 密码加密存储，不在前端和日志输出明文。
- 删除和编辑都要求唯一定位到 `database + table + timestamp`。
- 不会自动连接任何真实数据库；需要你在页面中显式配置连接后才会访问。

## 开发命令

```bash
pnpm install
pnpm dev
```

## 环境变量

复制 `.env.example` 为 `.env` 后可按需修改：

- `JWT_SECRET`
- `APP_SECRET`

如果未配置以上密钥，后端会在本地 `data/` 目录生成 `.secret` 文件。该目录已被 `.gitignore` 排除。

不要把真实数据库密码、Token 或服务器凭据提交到项目中。
